# BizControl  
### Business Appointment & Client Management System

BizControl is a backend system designed to help small businesses and independent professionals manage appointments, clients, and services efficiently.

## Overview
BizControl provides a centralized solution for managing scheduling-related business operations.  
The system is designed with scalability, clarity, and maintainability in mind.

---

## Core Features
- User management with role-based access (Admin / Client)
- Appointment scheduling with conflict prevention
- Client management
- Service management
- RESTful API
- Database-driven architecture

---

## Tech Stack
- Python 3.12
- FastAPI
- SQLAlchemy
- Alembic
- PostgreSQL
- Docker & Docker Compose
- OpenAPI / Swagger

## Development Status
The project is in an early development stage.  
Current focus is on implementing core business logic, authentication, and data integrity.
